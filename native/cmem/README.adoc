= cmem

Simple example for building a native executable with gradle

[source, shell]
----
$ ./gradlew installCmemExecutable
$ ./build/install/cmem/cmem
----

[source, shell]
----
$ gcc -Wall -Wpedantic -o test-mem test-mem.c
$ clang -Wall -Wpedantic -o heap-mem test-mem.c
----
